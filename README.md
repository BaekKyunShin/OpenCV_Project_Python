파이썬으로 만드는 OpenCV 프로젝트
==========================

## 스터디 개요

- 스터디 목적은 캐글 컴퓨터 비전 대회에 참여하기 위함입니다.
- '파이썬으로 만드는 OpenCV 프로젝트(이세우 저)' 교재를 활용하여 스터디를 했습니다.
